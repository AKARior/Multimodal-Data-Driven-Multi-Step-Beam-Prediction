# Model Checkpoints

The pretrained checkpoint files are available from the following Quark Cloud link:

https://pan.quark.cn/s/f259fc69e07a

Please download the checkpoint files and place them in this `weights/` directory.

Expected files:

```text
uav_side_numeric_beam_prediction_ckpt.pth
edge_side_image_beam_prediction_ckpt.pth
edge_multimodal_beam_prediction_ckpt.pth
```

These checkpoints correspond to the numeric-only, image-only, and multimodal fusion routes described in the main `README.md`.
